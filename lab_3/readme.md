# LR3 - Разделяемая память и семафоры

## Вариант 17

Родительский процесс создает два дочерних процесса. Пользователь вводит строки произвольной длины в родительский процесс, которые распределяются по правилу:
- Строки длиной > 10 символов → отправляются во второй дочерний процесс  
- Строки длиной ≤ 10 символов → отправляются в первый дочерний процесс

Дочерние процессы удаляют все гласные буквы из полученных строк и сохраняют результат в файлы.

**Новое требование:** Взаимодействие между процессами должно осуществляться через разделяемую память (shared memory) и семафоры для синхронизации доступа. Использовать именованные shared memory объекты и семафоры с уникальными именами.
