# LR4 - Исполняемые файлы и библиотеки

Требуется создать динамические библиотеки, которые реализуют заданный вариантом функционал. Далее использовать данные библиотеки 2-мя способами:

- Во время компиляции (на этапе линковки)

- Во время исполнения программы (библиотеки загружаются в память с помощью интерфейса ОС)

## Вариант 5

Расчет интеграла функции sin(x) на отрезке [A, B] с шагом e:

Сигнатура: float sin_integral(float a, float b, float e)

- Реализация №1: Метод прямоугольников

- Реализация №2: Метод трапеций

Расчет значения числа e (основание натурального логарифма):

Сигнатура: float e(int x)

- Реализация №1: (1 + 1 / x)^x

- Реализация №2: Сумма ряда по n от 0 до x: 1 / n!
